搭建个人博客

2022-10-18 2 min read # 搭建

先创建github的账号。

之后新建资料库

这边要输入用户名加上.github.io
勾选 add a readme file
然后点create repository

进入设置中的 developer settings


generate new token

照着图片填写，repo那边6个勾是一定要勾的。

记住最后给你的令牌😎

在gridea中远程里如图填写
域名：Github Pages 对应的域名（例如：https://Suremotoo.github.io 或 https://suremotoo.site）
仓库：你的静态文件存放的仓库（例如：http://Suremotoo.github.io ）分支：你的 Github Pages 对应的分支（例如：master 或 gh-pages）
用户名：仓库所属用户的用户名（通常就是你的 Github 账号的昵称，例如：Suremotoo）
邮箱：Git 推送时使用的邮箱（通常就是你的 Github 账号的邮箱，例如：suremotoo@163.com）
Token: Git 推送时需要的 Token，用来向 Github 提交构建后的文件 这里的 token 就是 第三步 申请的 Github personal access tokens
CNAME: 可通过这个【可选项】配置你自己的域名（例如：suremotoo.site）

点击检测远程连接
如果不行可能填写错误
如果填写正确也有很大概率会连接错误，多试几次😎

进入左栏文章，点击右上角的＋号，随便写点什么，再点击√保存。

最后点击左边的同步，多试几次低概率成功，同步成功后刷新页面就可以看到新的内容了~
还可以更改页面的其他内容，多多研究吧😎
